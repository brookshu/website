Work Experience

Northwestern Research | Design Automation of Intelligent Systems Lab 10/2023 – Present

At the start, I conducted research addressing the missing modality problem in time series applications, with a focus on wearable technologies. I performed literature surveys to explore solutions to this challenge and leveraged Meta’s AV-HuBERT technology, which uses audio and video data to generate captions, to understand current state-of-the-art approaches for handling missing/optional modalities in time series data.

Now, I am contributing to a confidential workplace safety report for a Fortune 500 company, where I lead research on advanced on-body and ambient sensing technologies. This involved conducting a comprehensive literature review on current workplace safety monitoring technologies as well as testing commercial off-the-shelf devices, such as iPhones, AirPods, and Meta Ray-Bans, to extract sensor data and evaluate their potential for enhancing workplace safety monitoring. We chose Apple products to work with MobilePoser, an open-source, real-time, full-body pose estimation system which works by leveraging data from device IMUs.

Intramotev Inc 2025 Summer

In the first half of the internship, I Developed an interactive Grafana dashboard with a Flask backend to process and visualize perception data, including camera, LIDAR, and GPS data. This involved creating a data pipeline to handle real-time sensor data. As this happened, the company was also migrating a legacy binary sensor data pipeline to the cloud-native InfluxDB time-series architecture, enabling real-time, scalable access to historical and live sensor data for analysis and visualization, so I helped with that as well, although I wasn't the primary engineer on the project. In the latter half of the internship, I created a pipeline to detect switch state signals. This involved using YOLOv11 model to detect the lights. After doing so, I employed Hough transforms, clustering (DBSCAN), and color masking to verify the state of the switch.

Deepsight Technology 2024 Summer

I developed a reference model for the company's ultrasound algorithm with documentation to facilitate collaboration and development. This consisted of interpreting high-level ultrasound code in C++, CUDA, Protobuf, and MATLAB to learn the geometry and data structures, as well as the preprocessing, beamforming, and post processing algorithms that the company uses. The program did not require a GPU to run for increased portability, meaning I had to circumvent parts of the company ultrasound code and rewrite them. In addition, I collaborated with engineers in all disciplines to align with other developments, optimize the efficiency of the program, and create the most comprehensible documentation for easy maintainability of the reference model.

Washington University | Cyber-Physical Systems Lab 2023 Summer

I created a testbed system to verify the accuracy and efficiency of an offloading model. The system ran a weak object detection model on a Raspberry Pi and offloaded to a stronger model on a more resource-abundant server. This setup aimed to emulate a realistic edge computing environment with constrained local processing capabilities and powerful edge resources. During the course of this project, I acquired experience with socket programming, TensorFlow Lite and Pytorch models, gaining inference time of various object detection models, and live detection using the cv2 library.

I also wrote a computer vision tutorial detailing how to create a custom dataset to train a model for object detection, which was used in WashU PhD course CSE 521 by Professor Chenyang Lu. In addition, I contributed to “Optimizing Edge Offloading Decisions for Object Detection” published in the 9th ACM/IEEE Symposium on Edge Computing (SEC 2024). Lastly, as part of the Research Experience for Undergraduates (REU) program, I attended weekly seminars on various computer science topics and interacted with undergraduate students, graduate students, and professors working on other REU projects.
